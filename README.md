# 量子化学小程序 – 软件使用说明

## 1. 软件概述

本软件是为 **Gaussian** 量子化学计算软件设计的辅助工具，主要提供以下功能：

- **能量/频率/坐标提取**：批量从 Gaussian 输出文件（`.log` / `.out`）中提取单点能、频率、热力学修正量、CBS‑QB3 能量、G4 能量、坐标等信息，并导出为 Excel（`.xlsx`）或 Word（`.docx`）文件。
- **文件信息查询**：查看 Gaussian 输出文件中的优化步数、特征值、收敛情况等详细信息，支持正则表达式搜索。
- **输入文件转换**：基于预定义或自定义模板，批量生成 Gaussian 输入文件（`.gjf`），支持 IRC、过渡态优化、单点能、自旋测试等多种任务类型。
- **文件重命名**：批量修改文件夹内的文件名（添加/删除/替换/大小写转换/序列化），支持预览和撤销操作。
- **多语言与主题**：支持中文/英文界面切换，内置多套界面主题。

软件采用 PyQt5 编写，适用于 Windows 系统（已提供启动脚本 `app.py`）。

## 2. 安装与运行

### 2.1 环境要求

- 操作系统：Windows 10 / 11
- Python 版本：3.8 及以上（如果直接运行源代码）
- 依赖库：PyQt5, pandas, docx, openpyxl, numpy 等（见 `requirements.txt`）

### 2.2 快速启动

1. 解压软件包至任意目录（例如 `D:\QuantumTools`）。

2. 确保已安装 Python 及所需依赖（或使用打包后的 `.exe` 文件）。

3. 双击 `app.py` 或在命令行执行：

   ```bash
   python app.py
   ```

4. 软件已打包为可执行文件的压缩包，直接双击 `QuantumApp.exe` 即可。

## 4. 保存模块（提取数据）

**用途**：从指定文件夹内的所有 Gaussian 输出文件（`.log` / `.out`）中批量提取能量、频率、坐标等信息，并导出为 Excel 或 Word 文件。

### 4.1 界面布局

- **输入文件**区：选择目标文件夹（支持历史记录和拖拽）。
- **显示区**：设置保存文件名、正则匹配字符串（高级功能），以及各类提取按钮。
- **输出记录**：显示处理日志。

### 4.2 操作步骤

1. **选择文件夹**  
   - 点击“打开文件夹”按钮，或直接拖拽一个文件夹到“搜索文件夹”下拉框中。
   - 软件会记住最近使用的 10 个文件夹。

2. **设置保存文件名**（可选）  
   - 在“保存文件名”下拉框中输入或选择输出文件的名称（不含扩展名）。  
   - 扩展名由提取功能自动添加（`.xlsx` 或 `.docx`）。

3. **设置正则匹配（高级）**  
   - “正则匹配1”和“正则匹配2”用于自定义提取规则（例如提取特定能量值），一般用户无需修改。

4. **选择提取内容**  
   点击下方任一按钮即可开始处理：

| 按钮                 | 提取内容                                | 输出格式 | 表头说明                                                     |
| -------------------- | --------------------------------------- | -------- | ------------------------------------------------------------ |
| 提取单点能/xlsx      | 最后一步的 HF 能量                      | `.xlsx`  | 物种名，能量（Hartree）                                      |
| 提取能量/xlsx        | 零点能、热修正、焓、自由能等            | `.xlsx`  | 物种名，ZPE，Cor_U，Cor_H，Cor_G，HF，E，U，H，G             |
| 提取频率/docx        | 所有振动频率（cm⁻¹）                    | `.docx`  | 物种名，频率列表（逗号分隔）                                 |
| 提取坐标/docx        | 最后一个优化步的笛卡尔坐标              | `.docx`  | 物种名，坐标（元素，X，Y，Z）                                |
| 提取更多信息/xlsx    | 转动常数、分子量、自由度等              | `.xlsx`  | 物种名，RC_x_GHz，RC_y_GHz，RC_z_GHz，RC_x_cm⁻¹，RC_y_cm⁻¹，RC_z_cm⁻¹，MW，num_freq |
| 提取CBS-QB3/xlsx     | CBS‑QB3 能量                            | `.xlsx`  | 物种名，CBS_Energy                                           |
| 提取G4/xlsx          | G4 能量                                 | `.xlsx`  | 物种名，G4_Energy                                            |
| 正则提取/高级功能    | 自定义正则表达式匹配                    | `.xlsx`  | 物种名，Match1，Match2（需预先设置正则）                     |
| 提取分步CBS-QB3/专用 | MP4，CCSD(T)，MP2，HF，CBS‑Int 等中间项 | `.xlsx`  | 物种名，MP4，CCSD(T)，MP2，MP4，HF，Int，OIii，T1，E         |

5. **查看结果**  
   - 提取完成后，表格区域会显示数据预览。  
   - 若勾选了“自动打开文件”，系统会用默认程序打开生成的文档。  
   - 日志区域会记录成功或失败的文件。

6. **清空表格 / 清空日志**  
   - 使用“清空表格”按钮仅清除表格内容，不清除表头。  
   - 使用“清空记录”按钮清空日志区域。

> **提示**：若文件夹中包含大量文件，提取速度取决于文件大小，请耐心等待。

## 5. 查询模块（搜索文件信息）

**用途**：查看单个 Gaussian 输出文件的详细优化过程，包括每一优化步的四个收敛判据、特征值、特征向量等，也支持正则表达式搜索。

### 5.1 界面布局

- **输入文件**区：选择目标文件（支持拖拽）。
- **正则匹配**：可自定义正则表达式搜索文件内容。
- **结果显示**：分为“现有功能结果”和“正则匹配结果”两个文本框。

### 5.2 操作步骤

1. **选择文件**  
   - 点击“打开文件”按钮，或直接拖拽一个 `.log` / `.out` 文件到“文件路径”输入框中。

2. **基本查找**  
   - **精简查找**：显示优化步的序号、前两个特征值、前三个特征向量和四个收敛状态（YES/NO）。  
   - **详细查找**：显示更多信息（四个特征值、四个特征向量、八个收敛判据数值）。

3. **正则搜索**  
   - 在“正则匹配”下拉框中输入或选择正则表达式（例如 `Zero-point correction=\s+([\d.-]+)`）。  
   - 点击“正则搜索”，匹配到的内容会显示在右侧“正则匹配结果”文本框中。

4. **清空结果**  
   - 点击“全部清空”可同时清空两个结果区域。

> **注意**：查询功能仅处理**单个文件**，不支持批量。

## 6. 转换模块（生成 Gaussian 输入文件）

**用途**：基于模板批量生成 Gaussian 输入文件（`.gjf`），支持从输出文件或已有输入文件中读取坐标和电荷/自旋多重度。

### 6.1 界面布局

- **输入文件**区：选择转换源（一个文件夹或单个文件）。
- **模板编辑**区：选择转换类型（IRC、高水平单点、自旋测试等），设置前后缀，选择计算模式，编辑/导入模板。
- **状态栏**：显示当前转换参数。
- **输出记录**：显示转换日志。

### 6.2 操作步骤

#### 6.2.1 选择源文件

- **转换文件夹**：勾选后点击“选择文件夹”按钮，软件会处理文件夹内所有符合条件的文件（读取类型由“读取文件类型”决定）。  
- **转换单文件**：勾选后点击“选择文件”，处理单个 `.log` / `.out` / `.gjf` 文件。

#### 6.2.2 设置转换参数

| 区域               | 选项                   | 说明                                                |
| ------------------ | ---------------------- | --------------------------------------------------- |
| **请选择功能**     | 更换水平(TS/OPT)       | 将 TS 或 OPT 结构的输出文件转换成指定水平的输入文件 |
|                    | 全IRC(TS)              | 为过渡态生成双方向 IRC 输入文件                     |
|                    | 两端IRC(TS)            | 同时生成前向和后向的两个 IRC 输入文件               |
|                    | 自旋测试(TS/OPT)       | 生成一系列不同自旋多重度的输入文件，默认最高自旋20  |
|                    | 高水平单点(TS/OPT)     | 生成高水平的单点能计算的输入文件                    |
|                    | 扫描(Mod)              | 为柔性扫描生成每一帧的输入文件                      |
|                    | IRC(IRC)               | 为 IRC 路径上的每个点生成独立输入文件               |
| **读取文件类型**   | GauOutFile / GauInFile | 选择从输出文件还是输入文件中读取坐标和电荷/自旋     |
| **增加文件前缀名** | 文本框                 | 生成的文件名前缀（例如 `opt`）                      |
| **增加文件后缀名** | 下拉框/文本框          | 生成的文件名后缀（例如 `irc`）                      |
| **计算模式**       | 模式1～模式5           | 预定义的计算水平（可自行导入新模式）                |
| **模板文件**       | 下拉框                 | 对应每种功能的模板文件（`.txt`），可编辑和保存      |

#### 6.2.3 编辑模板

- 模板内容显示在右侧文本编辑区，可以直接修改。
- 点击“保存模板”将当前内容保存到所选模板文件中。
- 点击“载入新文件”可导入外部 `.txt` 文件作为模板。
- 点击“载入新模式”可导入一个包含多个模板文件的文件夹（文件夹名会出现在“计算模式”下拉框中）。

#### 6.2.4 执行转换

- 确认状态栏显示的正确后，点击“生成文件”。
- 若选择“转换文件夹”且勾选了“自动打开文件”，转换完成后会打开目标文件夹。

> **模板占位符说明**：  
>
> - `replace-name` → 替换为实际文件名（含前后缀）  
> - `replace-coordinate` → 替换为坐标列表  
> - `replace-charge` → 替换为电荷  
> - `replace-multiplicity` → 替换为自旋多重度

## 7. 重命名模块（批量修改文件名）

**用途**：批量修改文件夹内所有文件的文件名（支持添加、删除、替换、大小写转换、序列化），提供预览和撤销功能。

### 7.1 界面布局

- **功能区**：选择文件夹，设置修改位置和操作类型。
- **文件显示**：表格展示原文件名、新文件名和状态（冲突提示）。
- **输出记录**：记录重命名操作日志。

### 7.2 操作步骤

#### 7.2.1 选择文件夹

- 点击“打开文件夹”按钮，或直接拖拽文件夹到“文件路径”下拉框中。

#### 7.2.2 设置修改参数

| 控件                                              | 说明                                                        |
| ------------------------------------------------- | ----------------------------------------------------------- |
| 位 置 1 / 位 置 2                                 | 指定文件名中的字符区间（从 0 开始计数）                     |
| 文件名之前 / 文件名之后 / 文件名中间 / 文件名前后 | 指定修改作用于文件名的哪一部分（==暂时未实现==）            |
| 增加 / 删除 / 替换 / 大小写 / 序列化              | 选择操作类型                                                |
| 更改内容                                          | 输入要添加、替换或插入的文本（序列化时支持 `{idx}` 占位符） |

**各操作的含义**：

- **增加**：在指定位置插入文本（不删除原有字符）。
- **删除**：删除位置1到位置2之间的字符。
- **替换**：将位置1到位置2之间的字符替换为新文本。
- **大小写**：将区间内的字母大小写互换。
- **序列化**：在指定位置插入序号（如 `{idx}` → `1`，`2` …），单个文件时固定为 `1`。

#### 7.2.3 预览与执行

- 若勾选“是否自动更新表格”，每次修改参数后表格会自动刷新预览。
- 也可点击“手动更新”按钮刷新表格。
- **状态列**会提示“冲突”（如果多个文件的新文件名相同）。
- 确认无误后点击“重命名”执行修改。

#### 7.2.4 撤销操作

- 软件在执行重命名前会自动备份文件夹内容到 `BackupFiles` 目录。
- 点击“撤销重命名”即可恢复到最近一次重命名前的状态。

#### 7.2.5 表格操作

- “自动调整表格”优化列宽。
- “清空表格”仅清空表格内容，不删除文件。

> **警告**：撤销仅支持**最近一次**重命名，多次操作后只能撤销最后一次。

## 9. 拖拽操作说明

软件支持将文件或文件夹直接拖放到主窗口，拖放行为取决于当前激活的页签：

| 页签   | 拖放对象                      | 行为                       |
| ------ | ----------------------------- | -------------------------- |
| 保存   | 文件夹                        | 设置为“搜索文件夹”         |
| 保存   | `.xlsx` / `.xls` 文件         | 设置为“保存文件名”         |
| 查询   | `.log` / `.out` 文件          | 设置为“文件路径”           |
| 转换   | 文件夹                        | 设置为“转换文件夹”         |
| 转换   | `.gjf` / `.log` / `.out` 文件 | 设置为“转换单文件”         |
| 转换   | `.txt` / `.md` 文件           | 将文件内容加载到模板编辑区 |
| 重命名 | 文件夹                        | 设置为“文件路径”           |
| 设置   | 任何对象                      | 不支持拖拽（会提示）       |

> 拖拽支持可在“设置”中关闭。

## 10. 常见问题

### Q1：提取能量时表格为空，怎么办？

- 检查文件夹路径是否正确，且文件夹内包含 `.log` 或 `.out` 文件。
- 检查“仅当前文件夹”是否误选（若文件在子文件夹中，需设置为“包含子文件夹”）。
- 查看日志区域的具体错误提示。

### Q2：转换生成的 `.gjf` 文件坐标不正确？

- 确认“读取文件类型”选择正确（输出文件选 `GauOutFile`，输入文件选 `GauInFile`）。
- 检查模板文件中的占位符是否完整（`replace-coordinate` 等）。
- 对于 IRC 任务，确保源文件确实包含 IRC 计算。

### Q3：重命名时状态列显示“冲突”怎么办？

- 冲突表示多个文件的新文件名相同，请修改参数（如扩大删除范围、添加序列化等）避免重名。

### Q4：当界面候选框显示出现问题，即配置文件有问题？

- 请删除安装目录的quantum_config.ini配置文件，重新启动会初始化配置文件

### Q5：如何添加新的模版模式（模板文件夹）？

- 在“转换”页面，点击“载入新模式”，选择包含多个模板文件（`.txt`）的文件夹。  
- 文件夹名称会出现在“计算模式”下拉框中，模板文件名需符合软件内置命名（如 `IRC-template.txt`，`OPT-template.txt` 等）。
